# 用户登录模块 PRD

## 1. 功能概述

### 1.1 模块简介
用户登录模块是系统的核心安全入口，提供用户身份认证功能。

### 1.2 用户角色
- 访客（未登录用户）
- 注册用户
- 管理员

## 2. 功能需求

### 2.1 登录功能
- 用户可以通过用户名和密码登录
- 支持记住登录状态（7天有效）
- 连续3次密码错误，账户锁定30分钟

### 2.2 注册功能
- 用户可以自主注册账号
- 用户名：4-20个字符，字母数字下划线
- 密码：至少8位，必须包含大小写字母和数字

### 2.3 找回密码
- 通过注册邮箱找回密码
- 发送重置链接到邮箱
- 重置链接30分钟内有效

## 3. 边界条件

### 3.1 用户名输入
- 空用户名 → 提示"用户名不能为空"
- 少于4个字符 → 提示"用户名至少4个字符"
- 超过20个字符 → 提示"用户名最多20个字符"
- 包含特殊字符 → 提示"仅支持字母、数字、下划线"

### 3.2 密码输入
- 空密码 → 提示"密码不能为空"
- 少于8位 → 提示"密码至少8位"
- 不含大写字母 → 提示"密码必须包含大写字母"
- 不含小写字母 → 提示"密码必须包含小写字母"
- 不含数字 → 提示"密码必须包含数字"

## 4. 安全要求

### 4.1 密码加密
- 密码必须加密存储，使用BCrypt算法

### 4.2 防暴力破解
- 单IP每分钟最多尝试60次登录
- 账户锁定期间拒绝所有登录请求

### 4.3 敏感信息
- 密码在传输和存储中不得明文
- 错误提示不得暴露账户是否存在
